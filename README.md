## Technical Stack

* React
* Redux
* React-Router
* Redux-Saga
* Ant-D

### Software Installation :
* Install node and npm 
* Install IntelliJ IDEA Ultimate Edition （Optional）
* Install Docker Desktop



## Install Dependencies:
```hash
$ npm install
```


## Running Locally:
```hash
$ npm run start
```

Visit: `http://localhost:3000/`

## Build
* Minimum 13.2G free  memory in your PC
```hash
$ npm run build
```
